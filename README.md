# CXXAudioRingBuffer

A lock-free SPSC ring buffer supporting non-interleaved audio.

> [!NOTE]
> C++17 is required.

## Installation

### Swift Package Manager

Add a package dependency to https://github.com/sbooth/CXXAudioRingBuffer in Xcode.

### Manual or Custom Build

1. Clone the [CXXAudioRingBuffer](https://github.com/sbooth/CXXAudioRingBuffer) repository.
2. `swift build`.
